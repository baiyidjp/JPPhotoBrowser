# JPPhotoBrowserDemo
Browse picture like WeChat.

> 使用 CocoaPods

```
pod 'JPPhotoBrowser' 

在使用的页面中 引用 #import "JPPhotoBrowserManager.h"

```

> 下载使用

```
直接将下载文件中的 JPPhotoBrowser 文件夹拖入项目中

在使用的页面中 引用 #import "JPPhotoBrowserManager.h"
```
